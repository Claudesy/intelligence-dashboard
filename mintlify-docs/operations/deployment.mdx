# Deployment

AADI deploys to **Railway** using Nixpacks with Node 22.

## Railway Configuration

The deployment is configured via `railway.toml`:

```toml
[build]
builder = "nixpacks"

[deploy]
startCommand = "NODE_ENV=production tsx --conditions react-server server.ts"
```

## Environment Variables

Ensure all required environment variables are set in Railway dashboard. See [Setup Guide](/operations/setup) for the full list.
