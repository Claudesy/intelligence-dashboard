# Privacy & Data Protection

AADI handles sensitive patient health information (PHI). All data practices comply with Indonesian healthcare data regulations.

<Note>
  For the full privacy policy, see the internal [PRIVACY.md](https://github.com/sentra-healthcare/abyss-monorepo/blob/main/app/primary-healthcare/dashboard/docs/PRIVACY.md).
</Note>

## Security Measures

- **Cookie-based auth** — Secure session management without third-party auth
- **CORS restrictions** — Only approved origins can access the API
- **Socket.IO auth middleware** — All WebSocket connections verified
- **Role-based access** — Profession-based permissions (Dokter, Bidan, Perawat, etc.)
